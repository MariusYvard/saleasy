# Marketing

Produce marketing and content that fits the user's market and voice. Covers LinkedIn, editorial planning, copywriting, repurposing, lead magnets, positioning and SEO.

## Clarify first

Read `saleasy-profile.md` from the working folder. If no profile exists, run `saleasy-setup` first (one-line heads-up, then continue). If the user has not said which job they want, ask one short question and offer the choices: LinkedIn post, LinkedIn profile, editorial calendar, campaign, long-form copy, SEO, repurposing, lead magnet, positioning. Then follow the matching section of `references/playbooks.md`.

## Always

Use the offer, target persona, differentiation and voice from the profile to shape every output. Pull proof assets (case studies, metrics, testimonials) from the profile instead of inventing claims. Tie every piece to a buyer pain and a funnel stage. When the user gives a topic, propose the angle before writing the full draft, unless they asked for the draft directly. Respect the writing directives: neutral and factual, no superlatives, no em dashes, straight quotes, no Oxford comma, vary list lengths.

## Output

Deliver content ready to paste. For multi-piece work (calendar, campaign), create a file in the working folder. For a single post or short copy, return it inline. Publishing to a connected social tool is optional; otherwise deliver copy for manual posting. End with one concrete next step (publish, schedule, A/B variant, repurpose), never a generic summary.
