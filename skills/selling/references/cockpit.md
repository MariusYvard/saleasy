# Daily briefing and weekly review

The pilotage layer. Answer "what should I do now" each morning and review the week honestly.

## Morning briefing
Read the profile for role and weekly targets. Assemble a short, ranked plan from available sources: today's meetings from the calendar (flag each for prep), follow-ups due today and deals with no next step from the CRM or `saleasy-pipeline`, replies that need an answer from email, new signals captured by monitoring and targets versus progress this week. Output the 3 to 5 things that matter today, ranked, each with the reason and the job that handles it. End with the single first action to take now. This is a status glance; for a full deal-by-deal review hand off to the pipeline review.

## Weekly review
Compare the week against the profile targets. Report what moved (meetings, replies, deals advanced, content shipped), what stalled and 3 priorities for next week. Where outreach ran, include the positive reply rate and reply quality, not just volume. Surface upcoming renewals and expansion plays when there is a customer base. Keep it to one screen. Be honest about misses rather than averaging them away.

## Recurrence
If the user wants the briefing every morning, offer to set up a scheduled task (for example daily at 8am) rather than waiting to be asked.

## Rules
Pull real data from connected tools or `saleasy-pipeline`; never invent numbers. If a source is not connected, say what is missing and work with what is available. Prioritize ruthlessly over completeness.
